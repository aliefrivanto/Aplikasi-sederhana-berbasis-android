# Sikop - Aplikasi Berita Android

Aplikasi Android sederhana untuk menampilkan daftar berita kepada pengguna.

## Fitur
- Menampilkan daftar berita
- Detail berita
- Tampilan menggunakan RecyclerView
- Navigasi antar halaman

## Teknologi
- Java
- Android Studio
- RecyclerView
- XML Layout
